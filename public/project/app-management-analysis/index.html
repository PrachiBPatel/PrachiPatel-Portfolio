  1. The following statement is made by your manager. Based on the examples and discussion in Lecture 1, transform it in to a question that can be answered with data analytics. Make sure you discuss the logic and reasoning you use to transform it.

People are downloading our app more and more but our current users are using the app less and less frequently.

Answer:

There are many reasons for the provided statement. Such as, 1. It will occurs because of poor management for the application. 2. Application functionality is not convenient for the long term or to use in a timely manner. 3. There is a chance of sudden drop-off of the application while using. 4. The application functionality is useful for must users but there is a chance of not having user friendly user functionality or UI/UX.

To solve mentioned points, company can do a proper management and enhancement for the current application. Company can do a survey for to identify the issues. Company can improve UI/UX as well as security for their application and make it more convenient for the users to use on a timely manner. They can create additional features to the application for current users to use their application more frequent.
